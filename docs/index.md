## 本地怎么启动这个学习站

```bash
nvm use
npm install
npm run docs:dev
```

`nvm use` 会读取仓库里的 `.nvmrc`，当前推荐 Node 22。

启动后直接打开：

- `http://localhost:5173`

如果你想打成静态站：

```bash
npm run docs:build
npm run docs:preview
```
