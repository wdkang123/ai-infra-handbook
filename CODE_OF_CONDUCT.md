# Code of Conduct

这个仓库是一个面向学习和分享的 AI Infra 项目。  
我们希望讨论保持清晰、友善、具体，让不同背景的学习者都能参与进来。

## 我们鼓励的行为

- 用具体例子解释问题，而不是只说“这里不行”
- 尊重初学者的问题，优先帮助对方建立系统地图
- 在提出改动时说明学习价值、风险和验证方式
- 讨论代码和文档本身，不攻击个人
- 避免在公开 issue 或 PR 中粘贴真实 token、密钥、私有 endpoint 或敏感日志

## 不适合的行为

- 嘲讽、贬低或人身攻击
- 用攻击性、歧视性或骚扰性的表达参与讨论
- 在没有上下文的情况下要求大规模重写
- 明知会误导初学者，仍提交未经验证的命令或文档
- 公开发布他人的敏感信息

## 讨论原则

如果你不同意一个实现，建议按这个结构表达：

```text
我看到的问题是：
它会影响的学习路径或系统行为是：
我建议的替代方案是：
我会用这些命令验证：
```

这样可以让讨论更容易落到具体改进上。

## 处理方式

维护者可以根据情况：

- 提醒参与者调整表达方式
- 编辑或隐藏包含敏感信息的内容
- 关闭偏离主题或无法建设性推进的讨论
- 在严重情况下限制继续参与

这个 Code of Conduct 的目标不是制造压力，而是保护学习项目的讨论质量。
