## Prometheus 格式说明

| 指标前缀 | 来源 | 说明 |
|---|---|---|
| `vllm_` | vLLM engine | 引擎原生指标 |
| `inference_service_` | inference-service 自定义 | 服务级指标 |

关键 vLLM 指标（来源）：
- `vllm_num_requests_running` — 当前运行中请求数
- `vllm_num_tokens_total` — 累计生成 token 数
- `vllm_avg_prefill_latency_ms` — 平均 prefill 延迟（ms）
- `vllm_avg_decode_latency_ms` — 平均 decode 每 token 延迟（ms）
- `vllm_gpu_cache_usage_utilization` — GPU KV cache 利用率（0~1）

关键 `inference_service_` 指标：
- `inference_service_requests_total` — 累计收到的请求
- `inference_service_streaming_total` — 累计流式响应
- `inference_service_non_streaming_total` — 累计非流式响应
- `inference_service_errors_total{type}` — 按错误类型累计
