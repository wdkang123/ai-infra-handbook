# Public Workbench

这个目录保留项目从研究、任务拆解、实现准备到评审收敛的过程材料。  
它不是新读者的第一入口；第一次学习请优先看 `README.md`、`docs/` 和 `projects/`。

公开保留这些材料的目的：

- 让读者看到一个学习型 AI Infra 项目如何被拆成可执行任务
- 让贡献者理解已有研究、任务卡、验收标准和评审口径
- 让后续路线图能从历史决策和复盘中继续生长

目录大致分为：

- `ready/`：待执行或曾经用于执行的任务卡
- `accepted/`：已接受的任务产物、资料包、蓝图和复盘材料
- `review/`：评审记录和修改建议
- `review-pending/`：待整理或待进一步评审的材料

这些文件已经按公开分享标准做过密钥和个人信息检查。贡献新任务时，请继续使用占位值，不要提交真实 API key、个人路径、内部 endpoint、敏感日志或私人账号。
